📈 4–20 mA Basics

Simple, real-world explanation for field use.

What is 4–20 mA?

A current signal used to represent a measurement.

  • 4 mA = 0%
  • 20 mA = 100%
Loop Diagram (Real Setup)
🔋 Power supply provides 24V to drive the loop

🎛️ Transmitter (TX) controls the current (4–20 mA)
based on the process variable

🧠 PLC / DCS reads the current and converts it
into engineering units (psi, level, flow, etc.)

🔁 Current must return to the power supply to complete the loop
Real Field Example

Range: 0–100 psi

  • 4 mA → 0 psi
  • 12 mA → 50 psi
  • 20 mA → 100 psi
Quick Formula

% = (mA - 4) / 16 × 100

Common Mistakes
  • Thinking 0 mA = 0% ❌
  • Forgetting scaling ❌
  • Always verify LRV / URV ✅
Field Tip

12 mA = 50% every time. Doesn’t matter what the units are.
