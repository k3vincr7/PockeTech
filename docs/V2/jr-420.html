📈 4–20 mA Basics
What it is and how it actually works in the field
What is 4–20 mA?

A 4–20 mA signal is a current loop used to represent a measurement.

  • 4 mA = 0% (Low range)
  • 20 mA = 100% (High range)
Loop Diagram
🔋 Power Supply provides voltage to drive the loop

🎛️ Transmitter controls the current (4–20 mA) based on the process value

🧠 PLC / DCS reads that current and converts it into engineering units

🔁 Current must return to complete the loop
Real Field Example

Range: 0–100 psi

  • 4 mA → 0 psi
  • 12 mA → 50 psi
  • 20 mA → 100 psi
Quick Formula

% = (mA - 4) / 16 × 100

Field Tip

12 mA is always 50%. Doesn’t matter if it's pressure, level, or flow.
