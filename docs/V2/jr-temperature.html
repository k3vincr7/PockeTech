🌡️ Temperature (RTD & Thermocouple)
What it is and how it actually behaves in the field
The Core Idea

Temperature is NOT measured directly. It is converted into an electrical signal and then scaled.

🧪 RTD (Resistance Temperature Detector)
🔬 How it Works
Resistance increases as temperature increases
📊 Common Types
Pt100 → 100Ω at 0°C (most common)
Pt1000 → 1000Ω at 0°C (better noise resistance)
🔌 Wiring Types
2-wire → simplest, least accurate
3-wire → most common (compensates lead resistance)
4-wire → highest accuracy
  • Very accurate and stable
  • Slower response than thermocouples
⚡ Thermocouple (TC)
🔥 How it Works
Generates a small voltage (mV) from heat difference
📊 Common Types
Type K → most common, high temperature
Type J → lower temp, older systems
Type T → precise low temperature
  • Fast response
  • Handles very high temperatures
  • Less accurate than RTDs
🔁 Signal Behavior
🧪 RTD
Sends resistance (Ω)
⚡ Thermocouple
Sends millivolts (mV)
🎛️ Transmitter
Converts signal → 4–20 mA for the control system
Real Field Behavior
  • RTD → smooth and stable readings
  • TC → faster but can drift
  • Wrong sensor type → completely wrong values
Common Problems
❌ RTD Open Circuit
→ Reading goes HIGH
❌ RTD Shorted
→ Reading goes LOW
❌ TC Reversed
→ Negative or incorrect reading
❌ Wrong TC Type
→ Incorrect temperature scaling
❌ Loose / Bad Wiring
→ Jumping or unstable readings
Quick Troubleshooting
⚠️ Reading too high
→ Check for open circuit (RTD)
⚠️ Reading too low
→ Check for short circuit
⚠️ Reading jumping
→ Check wiring or grounding
⚠️ Reading doesn't match process
→ Verify RTD vs Thermocouple type
Field Tip

Always confirm sensor type BEFORE troubleshooting. RTD and Thermocouple behave completely differently.
