How to Use
  1. Hold the phone away from metal surfaces, motors, and magnets
  2. Tap Set Baseline to zero the reading
  3. Slowly sweep the phone near the solenoid coil or valve body
  4. Status lights up DETECTED when a magnetic field is present

Best results: hold within 1–3 inches of the coil. Works on solenoid valves, relay coils, contactors, and motor windings. Reduce sensitivity (Low) if nearby steel causes false positives.

Android (Chrome): Full µT readings via Magnetometer API. iPhone (Safari): Compass distortion mode — shows heading deviation in degrees instead of µT. Both modes detect field changes reliably.
