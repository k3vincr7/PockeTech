🛠️ Field Workflow
Calibration → Loop Check → Commissioning → Troubleshooting
1. Calibration (Foundation)
Verify Instrument
Correct tag, correct range (LRV / URV), correct type
Apply Known Input
Pressure, mA, temp, level simulation
Adjust Output
Zero (LRV) and Span (URV)
5-Point Check
0 / 25 / 50 / 75 / 100%
Reality Check
If it’s not linear → something is wrong
2. Loop Check (Signal Path)
Field → DCS
Inject signal → verify correct value at DCS
DCS → Field (AO / DO)
Command from DCS → verify output at device
Verify Polarity
4–20 not reversed, DO not backwards
Check Scaling
mA matches engineering units
3. Final Element Check
Control Valve (AO)
4–20 → positioner → air → valve movement
On/Off Valve (DO / XV)
DO → solenoid → air → open/close
Check Fail State
FO / FC / FL must match design
4. Commissioning Mindset
You are verifying SYSTEM behavior, not just instruments:
Does field device respond?
Does DCS read correctly?
Does logic behave properly?
Do interlocks / trips work?
5. Real Failure Scenarios
⚠️ Signal good, valve not moving → Air problem
⚠️ No signal → Wiring / I/O issue
⚠️ Wrong reading → Scaling / calibration
⚠️ Interlock not working → Logic / permissives
6. Troubleshooting Logic
Always follow the path:
Field Device → Wiring → Marshalling → I/O → DCS → Output → Final Element
Break the loop in sections
Don’t guess — isolate
Check simplest thing first
Air, power, wiring before blaming logic
Field Rule

The signal tells you WHAT is happening.
The process tells you IF it makes sense.
