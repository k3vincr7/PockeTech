Torque & Thread Seal Notes

  • Prefer the **manufacturer torque table**. Over-torque can distort manifolds and impulse fittings.
  • For tapered threads: apply sealant in the **tightening direction** and keep the first thread clean.
  • Avoid tape on small tubing/compression fittings; use proper ferrule fittings per spec.
  • Never mix imperial/metric threads. If in doubt, gauge it.

If a connection keeps weeping, inspect surface finish, replace ferrules, and confirm thread form.
